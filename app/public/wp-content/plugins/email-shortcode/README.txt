Create a Custom Shortcode for Default Message Template of Event Espresso.

== Description ==

Create a Custom Shortcode for Default Message Template of <strong>Event&nbsp;Espresso</strong>.

<strong>Functionalities : </strong>
 * Activate/Deactivate the E-Shortcodes
 * Edit the E-Shortcodes
 * Set Dynamic/Static Value in the E-Shortcodes

== Installation ==

1. Upload `ee-email-shortcode.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to Use E-Shortcodes =

Found a Activated shortcode in a Section of Message Template. ( Section - Which is Selected when E-Shortcode is saved. )

e.g.
Shortcode Name is <strong>Event Number</strong>.
Shortcode Slug is <strong>Event_Number</strong>.
So, Found <strong>Event_Number_eees</strong> in a Section of Message Template.

And select a E-Shortcode which you want to use in Message.
