This WordPress plugin extends the Event Espresso plugin with the Spanish translation.

== Description ==
> This plugin requires the <a href="http://eventespresso.com/pricing/">Event Espresso plugin</a>. <strong>Don't use Event Espresso? <a href="http://eventespresso.com/pricing/">Buy the plugin</a></strong> and start managing Events using this revolutionary plugin!

The [Event Espresso](http://eventespresso.com/pricing/) developers don't focus on translating
their plugin. Instead they concentrate on the core functionalities of the plugin itself.
Therefore there are regular updates of the Event Espresso plugin. However, after each
automatic update, the manually placed Spanish translations are deleted. This plugin fixes
that issue for all the Spanish WordPress and Event Espresso users!

> Este plugin requiere de <a href="http://eventespresso.com/pricing/">Event Espresso plugin</a>. <strong>¿No utilizas Event Espreso? <a href="http://eventespresso.com/pricing/?ap_id=clmkg">Compra este plugin</a></strong> y empieza a Gestionar Eventos en tu página wordpress de una manera revolucionaria!

Los desarrolladores de [Event Espreso](http://eventespresso.com/pricing/) no se han enfocado en la traducción de este plugin. En cambio, si lo hace para las funcionalidades de este plugin. Por ello, encontrarás que hay actualizaciones constantes, pero sin embargo, en cada actualización automática, si has subido manualmente una traducción, esta será borrada. Este plugin arregla este problema para todos los usuarios de habla Hispana Wordpress y usuarios de Event Espreso!

Este plugin está mantenido por [Closemarketing](http://www.closemarketing.es/)

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your
WordPress installation and then activate the Plugin from Plugins page.
